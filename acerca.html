Acerca de este proyecto

"Piedra, Papel o Poderes" es un juego web simple que mezcla el clásico juego de piedra, papel o tijera con elementos extra para hacerlo más interesante y divertido.

Este proyecto está diseñado para practicar programación web usando HTML, CSS y JavaScript, además de implementar lógica de juego y diseño responsivo.

Espero que disfrutes el juego y que te animes a explorar más en el mundo del desarrollo web.
